# investment-log

複数のAIモデルによる投資判断を記録し、設定された投票者全体の95%以上が
賛成した場合だけ新規投資を許可する実験プロジェクトです。

現段階は設計フェーズです。実資金は扱わず、再現可能なペーパートレードから
検証を始めます。
